# 1. LEJ

#### ENV

Create DB:

1. create DB name
2. collation = utf8mb4_unicode_ci


#### Update vendor

```$ composer update```

#### NPM install

```
// clear
$ npm cache verify  (or "$ npm cache clear --force")
$ rm package-lock.json
$ rm -rf node_modules

// install
$ npm i
```

#### RUN frontend

* Dev - ```$ npm run dev```
* Prod - ```$ npm run prod```
* Watch - ```$ npm run watch``` or ```$ npm run watch-poll```
